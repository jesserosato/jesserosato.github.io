Some projects that I've worked on recently

  • WICit (repo) is a project to help women more easily find stores that accept WIC. WICit was built with Code for Sacramento to help showcase the California Department of Public Health's awesome new open data portal. WICit is built on Node, Express and Angular, and uses the Bootstrap CSS framework.
  • I built lists.io with some friends/esteemed colleagues from Snippet and Bamboo Creative. Lists is just a really simple way to make and share lists. Lists of your favorite movies, lists of the best articles you found on the web, lists of things you have to do today. Just lists, that's it. The project even won 2nd prize at Cereal Hack 4! Lists.io is built on the Symfony 2 PHP framework, and uses Bootstrap and jQuery on the front-end.
